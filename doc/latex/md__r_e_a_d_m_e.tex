Miluphcuda is a smoothed particle hydrodynamics code

Check out the \href{https://christophmschaefer.github.io/miluphcuda/index.html}{\texttt{ documentation}}!


\begin{DoxyItemize}

\item miluphcuda can be used to model fluids and solids.
\item miluphcuda runs on single Nvidia G\+P\+Us with compute capability 5.\+0 and higher.
