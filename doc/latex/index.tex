miluphcuda is the cuda port of the original miluph code. miluphcuda can be used to model fluids and solids. miluphcuda runs on single Nvidia G\+P\+Us with compute capability 5.\+0 and higher.

Main features are


\begin{DoxyItemize}
\item S\+PH hydro and solid
\item self-\/gravity (via Barnes-\/\+Hut tree)
\item porosity models (P-\/alpha, epsilon-\/alpha, Sirono)
