99 // with I_1: first invariant of stress tensor
100 // J_2: second invariant of stress tensor
101 // A, B: DP constants, which are calculated from angle of internal friction and cohesion
102 // in material.cfg you need:
103 // friction_angle =
104 // cohesion =
105 #define DRUCKER_PRAGER_PLASTICITY 0
106 // (3) Mohr-Coulomb (MC) yield criterion
107 // -> yield strength is given by yield_stress = tan(friction_angle) \times pressure + cohesion
108 // in material.cfg you need:
109 // friction_angle =
110 // cohesion =
111 #define MOHR_COULOMB_PLASTICITY 0
112 // Note: DP and MC are intended for granular-like materials, therefore the yield strength simply decreases (linearly) to zero for p<0.
113 // Note: For DP and MC you can additionally choose (1) to impose an upper limit for the yield stress.
114 // (4) Pressure dependent yield strength following Collins et al. (2004) and the implementation in Jutzi (2015)
115 // -> yield strength is different for damaged (Y_d) and intact material (Y_i), and averaged mean (Y) in between:
116 // Y_i = cohesion + \mu P / (1 + \mu P / (yield_stress - cohesion) )
117 // where *cohesion* is the yield strength for P=0 and *yield_stress* the asymptotic limit for P=\infty
118 // \mu is the coefficient of internal friction (= tan(friction_angle))
119 // Y_d = cohesion_damaged + \mu_d \times P
120 // where \mu_d is the coefficient of friction of the *damaged* material
121 // Y = (1-damage)*Y_i + damage*Y_d
122 // Y is limited to <= Y_i
123 // Note: If FRAGMENTATION is not activated only Y_i is used.
124 // For this model, you need the following parameters in material.cfg:
125 // yield_stress =
126 // cohesion =
127 // friction_angle =
128 // cohesion_damaged =
129 // friction_angle_damaged =
130 // If you want to additionally model the influence of some (single) melt energy on the yield
131 // strength, then activate COLLINS_PLASTICITY_INCLUDE_MELT_ENERGY, which adds a factor
132 // (1-e/e_melt) to the yield strength, and include the following in material.cfg:
133 // melt_energy =
