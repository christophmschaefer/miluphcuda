1 Gravity merging test case for miluphcuda
2 ----------------------------------------

9 -----------------------------------------
10 
11 This is a test case of "gently" merging spheres, in a mass range at the transition of the strength and gravity regime.
12 The bodies have masses of 3e20 kg and 1e20 kg, respectively, and start from rest, directly next to each other.
13 The default settings are self-gravity + Collins strength model (including fragmentation with Grady-Kipp).
14 
15 -----------------------------------------
16 
17 **To run the test case:**
18 
19 1. Compile miluphcuda with the `parameter.h` file from this directory.
20  Don't forget to also adapt the Makefile to your system.
21 2. Unpack `initials/impact.0000.gz` and move it to this directory.
22 3. Adapt the start script `run.sh` to your system (path to CUDA libs and to miluphcuda executable) and execute it.
23 4. Wait for the simulation to finish (75 output files).
24 
25 The setup consists of ~50k SPH particles, with a runtime around 1h on most current GPUs (benchmarked on a GTX 970).
26 
27 -----------------------------------------
28 
29 **Check/visualize the results**
30 
31 You can visualize the simulation for example with *Paraview*. Find the latest release at https://www.paraview.org/.
32 First, enter the simulation directory and run
33 
34  utils/postprocessing/create_xdmf.py
35 which creates an xdmf file (*paraview.xdmf* by default). You can use the default cmd-line options,
36 so no need to set anything explicitly. Then start Paraview and either
37 
38 * directly open the created paraview.xdmf and choose settings yourself
39 * load the prepared Paraview state in *results/paraview.pvsm* (*File -> Load State*), and select
40  the created paraview.xdmf file under *Choose File Names*
41 
42 * compare results (e.g., visualized with Paraview) to the animations in `results/`, which show
43  the interior structure of the colliding bodies (cut views) for different quantities.
44 
45 * You can also visualize the workings of the Collins plasticity model by running `results/plot_plastic_yielding.sh`.
46  This produces plots for shear stress vs. pressure, including the respective yield limit curves.
