97 // (1) Simple von Mises plasticity with a constant yield strength, where you need in material.cfg:
98 // yield_stress =
99 // (2) Drucker-Prager yield criterion -> yield strength is given by the condition \sqrt(J_2) + A * I_1 + B = 0
100 // with I_1: first invariant of stress tensor
101 // J_2: second invariant of stress tensor
102 // A, B: Drucker Prager constants, which are calculated from angle of internal friction and cohesion
103 // in material.cfg you need:
104 // friction_angle =
105 // cohesion =
106 // Note: Drucker-Prager and Mohr-Coulomb are intended for granular-like materials, therefore
107 // the yield strength simply decreases (linearly) to zero for p<0.
108 // (3) Mohr-Coulomb yield criterion
109 // -> yield strength is given by yield_stress = tan(friction_angle) \times pressure + cohesion
110 // in material.cfg you need:
111 // friction_angle =
112 // cohesion =
113 // (4) Pressure dependent yield strength following Collins et al. (2004) and the implementation in Jutzi (2015)
114 // -> yield strength is different for damaged (Y_d) and intact material (Y_i), and averaged mean (Y) in between:
115 // Y_i = cohesion + \mu P / (1 + \mu P / (yield_stress - cohesion) )
116 // where *cohesion* is the yield strength for P=0 and *yield_stress* the asymptotic limit for P=\infty
117 // \mu is the coefficient of internal friction (= tan(friction_angle))
118 // Y_d = cohesion_damaged + \mu_d \times P
119 // where \mu_d is the coefficient of friction of the *damaged* material
120 // Y = (1-damage)*Y_i + damage*Y_d
121 // Y is limited to <= Y_i
122 // Note: If FRAGMENTATION is not activated only Y_i is used.
123 // For this model, you need the following parameters in material.cfg:
124 // yield_stress =
125 // cohesion =
126 // friction_angle =
127 // cohesion_damaged =
128 // friction_angle_damaged =

133 // (5) Simplified version of the Collins et al. (2004) model, which uses only the

138 // cohesion =
