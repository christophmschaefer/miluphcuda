1 ### Rotating sphere test case
2 
3 Standard SPH cannot handle rigid rotations.
4 This test case is one of the standard tests to check if the TENSORIAL_CORRECTION
5 implementation works.
6 
7 How to run it:
8 1. copy parameter.h to the root source directory of miluphcuda (usually cp parameter.h ../../)
9 2. compile
10 3. copy the binary to this directory
11 4. start run.sh in this directory
12 
13 The initial particle distribution is a solid sphere with R=1, rotating with omega_z=2 pi /100 c_s/R, where c_s denotes the
14 sound speed, which is set to 1.
15 
16 The test runs only for some minutes. In the end, either check the file **conserved_quantities.log** for the total
17 angular momentum (column 1 is time, column 12 is the absolute value of the total angular momentum of all particles), or
18 take a look at **angular_momentum.png**.
