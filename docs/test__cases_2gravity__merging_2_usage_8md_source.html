9 -----------------------------------------
10 
11 This is a test case of gently merging spheres, in a mass range at the interplay of strength and gravity.
12 The bodies have masses of 3e20 kg and 1e20 kg, respectively, and start from rest, just next to each other.
13 The default settings are self-gravity + Collins strength model (including fragmentation).
14 
15 -----------------------------------------
16 
17 **To run the test case:**
18 
19 1. Compile miluphcuda using the `parameter.h` file from this directory.
20  Don't forget to also adapt the Makefile to your system.
21 2. Unpack `impact.0000.gz`.
22 3. Adapt the start script `run.sh` to your system (path to CUDA libs and to miluphcuda executable) and execute it.
23 4. Wait for the simulation to finish (75 output files).
24  Output to stdout and stderr is written to `miluphcuda.output` and `miluphcuda.error`, respectively.
25 
26 -----------------------------------------
27 
28 **Check/visualize the results**
29 
30 You can visualize the simulation for example with *Paraview*. Find the latest release at https://www.paraview.org/.
31 First, enter the simulation directory and run
32 
33  utils/postprocessing/create_xdmf.py
34 which creates an xdmf file (*paraview.xdmf* by default). You can use the default cmd-line options,
35 so no need to set anything explicitly. Then start Paraview and either
36 
37 * directly open the created paraview.xdmf and choose settings yourself
38 * load the prepared Paraview state in *paraview.pvsm* (*File -> Load State*), and select
39  the created paraview.xdmf file under *Choose File Names*
